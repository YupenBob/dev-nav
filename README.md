## ✨ 特性

- 🔍 **快速搜索** — 实时过滤，找工具不再大海捞针
- 📚 **分类清晰** — 工具站 / 开源项目 / 官方文档 / 学习平台，多维度整理
- 🤖 **AI 工具专区** — 收录 ChatGPT、Claude、Cursor 等主流 AI 编程工具
- 🌐 **精选资源** — 每个站点均有简介，告别无脑收藏
- 📱 **响应式布局** — 移动端友好，随时查阅
